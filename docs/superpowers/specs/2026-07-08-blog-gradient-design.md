# 博客主题渐变增强设计

## 背景

当前博客封面、页头、导航栏、底部栏、网页背景与 OG 分享图使用渐变，但用户反馈渐变不够明显、颜色跨度偏小。本设计在保留现有蓝→蓝紫→暖色品牌方向的前提下，加大冷暖两端与中间层次的距离，使渐变更醒目。

## 目标

- 让浅色/深色模式下的渐变颜色跨度更大、层次更分明。
- 同步更新所有使用渐变的界面元素：
  - 首页全屏封面（`#page-header.full_page`）
  - 文章页顶部图（`#page-header.post-bg`）
  - 非首页页头（`#page-header.not-home-page`）
  - 无顶图时的导航栏背景（`#page-header.not-top-img`）
  - 滚动后固定的导航栏背景（`#page-header.nav-fixed #nav`）
  - 阅读模式下的文章页顶（`.read-mode #page-header.post-bg`）
  - 网页背景（`#web_bg`）
  - 底部栏（`#footer`）
  - 分享 OG 图自动生成的 SVG 渐变背景
- 保持文字与背景有足够对比度，避免内容可读性下降。

## 最终配色

### 浅色模式

- **主渐变**：`linear-gradient(135deg, #4facfe 0%, #6ec6ff 25%, #a18cd1 55%, #fbc2eb 75%, #ffd2a8 100%)`
- **固定导航栏/无顶图背景**（半透明）：`rgba(79,172,254,0.96)` → `rgba(161,140,209,0.96)` → `rgba(255,210,168,0.96)`
- **网页背景**：`linear-gradient(180deg, #f3f9ff 0%, #edf2ff 55%, #fff8f3 100%)`
- **遮罩渐变**：`linear-gradient(180deg, rgba(26, 73, 118, 0.08) 0%, rgba(60, 66, 122, 0.16) 100%)`
- **文字颜色**：保持白色 `#ffffff`，阴影加深以提升对比度。

### 深色模式

- **主渐变**：`linear-gradient(135deg, #020024 0%, #090979 25%, #1a2440 50%, #5b425d 75%, #9d4edd 100%)`
- **固定导航栏/无顶图背景**（半透明）：`rgba(2,0,36,0.96)` → `rgba(26,36,64,0.96)` → `rgba(157,78,221,0.96)`
- **网页背景**：`linear-gradient(180deg, #0a0a1a 0%, #11132e 55%, #1a1529 100%)`
- **遮罩渐变**：`linear-gradient(180deg, rgba(6, 12, 24, 0.2) 0%, rgba(6, 12, 24, 0.38) 100%)`
- **文字颜色**：保持 `#eaf4ff`，阴影保持深色。

## 修改范围

1. `themes/butterfly/_config.yml` 中 `inject.head` 内联样式段落。
2. `scripts/og-image.js` 中 `buildSvg` 的 `<linearGradient id="bg">` 定义。

## 验收标准

- 浅色/深色模式下，封面、页头、固定导航栏、底部栏、网页背景均使用新渐变。
- OG 图生成结果使用浅色主渐变。
- 文字（站点标题、副标题、文章标题、导航链接、底部版权）与新背景对比度可辨识。
- 本地 `hexo generate` 可成功构建。
